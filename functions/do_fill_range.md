```c
void DO_FILL_RANGE(int address, char value, int size)
```

#### Description

Fills a range in memory with a single value.

#### Parameters
  - `address` - the address to start filling from
  - `value` - the value to fill with
  - `size` - the size of the range to fill

#### Returns

n/a

#### Location

| Region      | Address      |
| ----------- | ------------ |
| NTSC-J      | `80004050`   |
| NTSC-U      | `80004050`   |
| PAL         | `80004050`   |
